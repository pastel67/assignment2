# 학생 관리 프로그램

학생 정보를 등록, 삭제, 점수 통계 및 우수생을 출력할 수 있는 학생 데이터 관리 프로그램입니다. 텍스트 파일을 활용하여 학생 데이터 관리와 다양한 통계 기능을 제공합니다.


# 파일별 기능 정리

- **bin/dart_application.dart** : 프로그램의 메인 실행 파일, 전체 메뉴 및 흐름 제어
- **lib/assignment_1.dart** : 필수 과제 기능(학생 목록 및 통계 출력)
- **lib/assignment_2.dart** : 도전 과제 기능(추가 기능 구현)
- **lib/best_student.dart** : 최고 점수 학생(동점자 포함) 출력
- **lib/delete_student_data.dart** : 학생 정보 삭제 기능
- **lib/input_student_data.dart** : 학생 이름/점수 입력 및 유효성 검사
- **lib/load_student_data.dart** : 학생 정보 파일에서 데이터 로드 및 리스트 생성
- **lib/new_student_data.dart** : 새로운 학생 정보 추가
- **lib/score.dart** : 학생 점수 데이터 클래스 정의
- **lib/total_score_average.dart** : 전체 평균 점수 및 평균 이상 학생 목록 출력
- **result.txt** : 프로그램 실행 결과 저장
- **students.txt** : 학생 이름과 점수 데이터 저장

# 구동 설명

- 첫 실행 메뉴

프로그램을 실행합니다.

메뉴를 선택하세요.

1. 필수 과제 // 1 을 입력하면 필수과제 프로그램 실행
2. 도전 과제 // 2 를 입력하면 도전과제 프로그램 실행
3. 종료 // 3 을 입력하면 '감사합니다.' 문구와 함께 프로그램 종료
   // 7 을 입력하면 이스터에그 출력과 함께 프로그램 종료

> 선택: // 메뉴 번호 입력


- 필수과제

  _등록된 학생 목록_
  [홍길동, 김철수, 현수진, 정희수, 노현호,
  , 박상현, 윤시현, 최고은, 홍길동(2), 홍길동(3),
  , 홍길동(4), 홍길동(5), 홍길동(6)]
  // students.txt 파일에 등록되어 있는 학생의 명단 출력
  // 5명씩 끊어서 출력하기 위해서 .map 을 사용하지 않고 기존 방식을 유지 했습니다.

  > 학생 이름: ex)홍길동 // 학생 이름 입력

  이름 : 홍길동, 점수 : 90
  이름 : 홍길동(2), 점수 : 39
  이름 : 홍길동(3), 점수 : 38
  이름 : 홍길동(4), 점수 : 23
  이름 : 홍길동(5), 점수 : 38
  이름 : 홍길동(6), 점수 : 58
  저장이 완료되었습니다. // 같은 이름을 가진 학생 데이터를 모두 불러와 저장 및 종료

  > 학생 이름: ex)홍길동(2)

  이름 : 홍길동(2), 점수 : 39
  저장이 완료되었습니다. // 번호 입력 시 해당 학생의 데이터만 불러와 저장 및 종료


- 도전과제

메뉴를 선택하세요.

1. 우수생 출력 // 최고점자 (동점자 포함) 데이터 출력
   우수생: 정희수 ( 평균 점수: 100)
   우수생: 노현호 ( 평균 점수: 100)
   우수생: 최고은 ( 평균 점수: 100)

2. 전체 평균 점수 출력 // 등록된 학생 전체 점수의 평균 출력 / 평균 점수 이상인 학생 출력
   
   전체 평균 점수: 65.08
   평균점 이상의 학생 목록을 보시겠습니까?(y/n) // y 입력 시 학생 목록 출력 / n 입력 시 이전 단계로 이동
   <전체평균 점수 이상 학생>
   홍길동
   김철수
   정희수
   노현호
   윤시현
   최고은

3. 학생 정보 검색 // 원하는 학생의 데이터 검색

   > 학생 이름: 홍길동 // 학생 이름 입력

   이름 : 홍길동, 점수 : 90
   이름 : 홍길동(2), 점수 : 39
   이름 : 홍길동(3), 점수 : 38
   이름 : 홍길동(4), 점수 : 23
   이름 : 홍길동(5), 점수 : 38
   이름 : 홍길동(6), 점수 : 58 // 입력한 학생 이름과 일치하는 데이터 모두 출력

4. 학생 데이터 등록 // 새로운 학생 데이터 등록
   
   등록할 학생의 이름을 입력해주세요.
   > 학생 이름: // 등록할 학생 이름 입력
   > 학생 점수: // 등록할 학생 이름 입력
   > 000학생의 정보가 저장되었습니다.

5. 학생 데이터 삭제 // 등록된 학생 데이터 삭제
   
   > 학생 이름: // 삭제할 학생 이름 입력
   > 학생의 정보가 삭제 되었습니다. // 등록된 학생 이름 삭제 / 동명이인일 경우 가장 마지막에 저장된 데이터부터 삭제

6. 이전 단계 // 현 단계 종료 후 이전 단계로 이동
7. 프로그램 종료 // 현 단계에서 프로그램 종료

> 선택: // 메뉴 번호 입력
